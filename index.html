Welcome to my 'File metadata MS' page

This is a microservice which returns the size and other metadata information corresponding to the uploaded files.

Choose files:

  • You can select one or multiple files to retrieve the metadata for.
  • In case of files of a large size, or if there are a lot of files, then the response might take longer.
  • Sample response:
    [{"name":"Pomodoro_clock.PNG","size":26746,"type":"image/png"},{"name":"Tic_Tac_Toe.PNG","size":41358,"type":"image/png"}]
